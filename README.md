# Pokémon Project Pokélachia (WIP)

This is a custom Pokémon hack game based on Pokémon Polished Crystal (https://github.com/Rangi42/polishedcrystal/) which is based on Pret's disassembly of Crystal (https://github.com/pret/pokecrystal).

Project Pokélachia seeks to create a brand new game, complete with a new region, new pokemon, and new story and characters set in the pokeworld equivalent of Appalachia.
